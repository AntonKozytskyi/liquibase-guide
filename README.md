# How to run application locally

## Database setup
## Create a database structure
First, we need to create a user (see resources/application.yml for the user name and password):
```sql
CREATE USER bloguser WITH PASSWORD 'p38ddRm74';
```

Second, we need to create a database and set our user as owner:
```sql
CREATE DATABASE blogdb OWNER bloguser;
```

Now we are ready to create a new table for articles:
```sql
CREATE TABLE IF NOT EXISTS article (
    id int8 NOT NULL GENERATED BY DEFAULT AS IDENTITY,
    title VARCHAR (120) NOT NULL,
    text VARCHAR (5000) NOT NULL,
    CONSTRAINT "articlePK" PRIMARY KEY (id),
    CONSTRAINT uc_articletitle_col UNIQUE (title)
);
```

## Run application
You can run application either in IntelliJ IDEA or via a terminal.

### How to run application via a terminal
```bash
mvn spring-boot:run
```
Please make sure that there are no errors in a terminal and the following log message is in the list:
> Tomcat started on port(s): 8080 (http) with context path ''

## How to open website in a browser
Follow the URL: [website](http://localhost:8080/)
